## Usage & Guidelines

<ul>
  <li>Buttons should indicate an action.</li>
  <li>Button labels should always be Title Case.</li>
  <li>They should be discoverable, easy to identify, and specific.</li>
  <li>
    Icons can be used alone or added to the left or right of the label. If used, the icon should
    signify what the Button does.
  </li>
  <li>Use icon-only variants in dense environments or when space is limited.</li>
  <li>
    When deciding which Button to use, consider the level of priority of the action, as well as how
    much visual emphasis the Button should have in the context of the page it will live on.
  </li>
</ul>
